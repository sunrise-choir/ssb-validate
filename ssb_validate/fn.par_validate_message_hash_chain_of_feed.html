pub fn par_validate_message_hash_chain_of_feed<T: AsRef<[u8]>, U: AsRef<[u8]>>(
    messages: &[T], 
    previous: Option<U>

    [T]: ParallelSlice<T>,
    T: Sync,
    U: Sync + Send + Copy, 

Batch validates a collection of messages, all by the same author, ordered by ascending sequence number, with no missing messages.

It expects the messages to be the JSON encoded message of shape: {key: "", value: {...}}

This will mainly be useful during replication. Collect all the latest messages from a feed you’re replicating and batch validate all the messages at once.
